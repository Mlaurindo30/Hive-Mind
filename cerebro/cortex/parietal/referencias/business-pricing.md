     9|
    10|## Como Defino Precos
    11|
    12|- **Modelo:** Por projeto + retainer mensal opcional
    13|- **Moeda:** BRL
    14|- **Base de calculo:** Complexidade do agente + custo de API estimado + horas de desenvolvimento + margem

    23|## Regras de Precificacao
    24|
    25|- **Margem minima:** 20% sobre custo total (horas + API + infra)
    26|- **Cada OKR carrega estimativa de custo:** tempo + API + receita esperada
    27|- **Checkpoint no meio do projeto:** burn vs. projecao
    28|- **Nunca dizer "nao tem problema" sem precificar primeiro**
