# Sinapse Agent — Codex CLI Integration

This vault is a Sinapse Agent memory layer. Use the `sinapse-memory` MCP tools:

- `sinapse_query` — search all memory backends (NeuralMemory, claude-mem, Graphify)
- `sinapse_save_decision` — save a decision to work/active/
- `sinapse_save_learning` — save a learning/insight to brain/Patterns.md
- `sinapse_health` — health check of all backends
- `sinapse_session_end` — end session, update brain/Current State.md

See [[AGENTS.md]] for vault conventions.
See [[CLAUDE.md]] for detailed operating manual.
